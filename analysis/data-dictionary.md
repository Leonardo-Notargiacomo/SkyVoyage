| Word  | Definition |
| ------------- | ------------- |
| Flight  | Airport to airport  |
| Trip  | Departure to Arrival  |
| Return trip  |  Arrival to Departure  |
| Outbound flight  | Departure to Transfer / Arrival  |
| Connecting flight  | Transfer to Transfer / Arrival  |
| Transit flight  | Connecting flight  |
| Transfer | Connecting flights; Between arrival and departure should be minimal 2.5 hrs|
| Flight time  | Actual duration (hours:minutes) from airport to airport  |
| Discount  | Static or Dynamic price reduction. Dynamic price reduction can be dependent on time (early bird, last minute), or on availability of plane (the fewer seats available, the higher the price), or on both.  |
